# nextjs-file-upload-typescript

## 事前準備

```shell
# create project
yarn create next-app --typescript

# install
yarn add @prisma/client
yarn add prisma --dev

# init
npx prisma init
```

.env
```
DATABASE_URL="file:./dev.db"
```

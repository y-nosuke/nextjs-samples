# nextjs-samples

## プロジェクト作成

### javascript

```shell
yarn create next-app (プロジェクト名)
cd (プロジェクト名)
yarn dev
```

### typescript

```shell
yarn create next-app (プロジェクト名) --typescript
cd (プロジェクト名)
yarn dev
```


## アクセス

http://localhost:3000

http://localhost:3000/api/hello
